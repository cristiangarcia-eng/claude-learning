# Project Brief: Orbit Task Manager -- Phase 1 Launch

## Overview
Orbit Task Manager is launching a new "Team Workspaces" feature that allows teams to create shared project spaces with customizable workflows. Phase 1 focuses on core workspace functionality for teams of 5-25 people.

## Goals
- Launch Team Workspaces to 100% of Pro and Enterprise customers
- Achieve 30% adoption within 60 days of launch
- Reduce the #1 support ticket category ("how do I share projects with my team") by 50%

## Target Users
- Team leads managing 5-15 direct reports
- Project managers coordinating cross-functional work
- Department heads who need visibility into team progress

## Scope
- Shared project spaces with invite-by-email
- Three default workflow templates (Kanban, Sprint, Simple List)
- Team-level permissions (Admin, Editor, Viewer)
- Activity feed showing team actions

## Out of Scope
- Guest access (Phase 2)
- Custom workflow builder (Phase 2)
- API access to workspaces (Phase 2)
- Integration with third-party tools

## Timeline
- Design: Complete
- Engineering: 6-week sprint starting March 3
- Beta: April 14 (internal + 20 design partners)
- GA: May 5
