# Hands-On Exercises

Practice what you've learned with real-world tasks. Each exercise is tied to a specific module — you learn the feature, then practice it with real data.

## How It Works

Each exercise has **two parts**:
1. **The task** — a realistic business challenge with real data files
2. **The feature** — you solve it by building something from the module you just learned (a slash command, a skill, a subagent, a hook, etc.)

You don't just use Claude Code — you extend it.

## Tips for Success

| Do | Avoid |
|----|-------|
| Complete the module before the exercise | Skipping to the exercise without learning the feature |
| Build the reusable artifact (command, skill, agent) | Just getting the task done without the feature |
| Test your artifact on different inputs | Hardcoding it for just this exercise's data |
| Read the "Connection to Module" table in each exercise | Ignoring which feature concept to practice |
