# Voice and Remote Control

Work with Claude from anywhere — speak instead of type, or continue sessions from your phone.

## Voice Dictation

Speak your requests instead of typing. Great for brainstorming, longer instructions, or when you're away from your keyboard.

```
/voice
```

- **Push-to-talk**: Hold the trigger key to speak, release to send
- **20 languages** supported
- Configure the trigger key with `/keybindings`

### When it's useful

- Brainstorming ideas out loud
- Dictating long descriptions or requirements
- Working hands-free while reviewing something on screen
- Describing visual changes you want to make

## Remote Control

Continue a Claude Code session from your phone, tablet, or any browser. The session stays running on your computer — you just control it remotely.

```bash
claude remote-control
```

1. Claude prints a URL and shows a QR code (press `spacebar`)
2. Scan the QR code with your phone or open the URL in any browser
3. You're connected — type messages and see responses in real time

### Use cases

- Review a report Claude created while you're in a meeting
- Continue a task from your phone while commuting
- Monitor progress from another device
- Share your session temporarily with a colleague

## Dispatch

Send tasks to Claude from your phone and let it work in the background.

1. Open the **Claude Desktop app** or go to **claude.ai/code**
2. Send a message describing the task
3. Claude starts working — you can close the app
4. Check back later for results

```
"Analyze the sales data from last month and create a summary report"
```

Claude works autonomously. When it's done, the results are waiting for you.

> **Tip**: Combine Remote Control with Voice Dictation — speak commands from your phone while Claude executes on your computer.
