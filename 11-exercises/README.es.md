Para todos. Las tareas son universales — lo que cambia es qué funcionalidad de Claude Code practicas.

| **Finanzas** | Categorización de gastos (#2), análisis de datos (#5), priorización de presupuesto (#8) |
| **RRHH** | Estandarización de taxonomías (#4), análisis de conversaciones (#5) |
| **Operaciones** | Evaluación de proveedores (#3), auditoría de carpetas (#9), integración de sistemas (#11) |

| **Marketing** | Investigación (#3), QA de contenido IA (#6), contenido end-to-end (#10) |
