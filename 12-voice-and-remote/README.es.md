# Voz y Control Remoto

Trabaja con Claude desde cualquier lugar — habla en vez de escribir, o continúa sesiones desde tu teléfono.

## Dictado por Voz

Habla tus solicitudes en vez de escribirlas. Genial para brainstorming, instrucciones largas, o cuando estás lejos del teclado.

```
/voice
```

- **Pulsar para hablar**: Mantén presionada la tecla de activación para hablar, suelta para enviar
- **20 idiomas** soportados
- Configura la tecla de activación con `/keybindings`

### Cuándo es útil

- Hacer brainstorming de ideas en voz alta
- Dictar descripciones largas o requisitos
- Trabajar manos libres mientras revisas algo en pantalla
- Describir cambios visuales que quieres hacer

## Control Remoto

Continúa una sesión de Claude Code desde tu teléfono, tablet o cualquier navegador. La sesión sigue corriendo en tu computadora — solo la controlas remotamente.

```bash
claude remote-control
```

1. Claude muestra una URL y un código QR (presiona `espacio`)
2. Escanea el código QR con tu teléfono o abre la URL en cualquier navegador
3. Estás conectado — escribe mensajes y ve respuestas en tiempo real

### Casos de uso

- Revisar un informe que Claude creó mientras estás en una reunión
- Continuar una tarea desde tu teléfono mientras vas en camino
- Monitorear el progreso desde otro dispositivo
- Compartir tu sesión temporalmente con un colega

## Dispatch

Envía tareas a Claude desde tu teléfono y déjalo trabajar en segundo plano.

1. Abre la **app de escritorio de Claude** o ve a **claude.ai/code**
2. Envía un mensaje describiendo la tarea
3. Claude empieza a trabajar — puedes cerrar la app
4. Vuelve después para ver los resultados

```
"Analiza los datos de ventas del mes pasado y crea un informe resumen"
```

Claude trabaja de forma autónoma. Cuando termina, los resultados te están esperando.

> **Consejo**: Combina Control Remoto con Dictado por Voz — habla comandos desde tu teléfono mientras Claude ejecuta en tu computadora.
